Motherboard

A motherboard is the central circuit board in a computer, connecting essential components like the CPU, RAM, GPU, and peripherals. It facilitates communication and coordination among these parts to make the computer function as a unified system.

GPU

A graphics card, or GPU is a specialized component in a computer that handles graphics rendering. It accelerates image and video processing, enhancing visual performance in tasks such as gaming, video editing, and graphic design. The GPU has its own dedicated memory and processing power, working in tandem with the CPU to deliver high-quality visuals on a display.

RAM

Random Access Memory or RAM is a type of computer memory that provides fast and temporary storage for actively running programs and data. It allows quick access to information for the CPU, enabling efficient multitasking and overall system performance. The more RAM a computer has, the better it can handle simultaneous tasks and applications.

CPU

The Central Processing Unit (CPU) is the core component of a computer responsible for executing instructions and performing calculations. Often referred to as the "brain" of the computer, the CPU interprets and processes data from software, managing tasks such as running applications, handling system functions, and executing program instructions. CPU performance is a key factor in determining overall computing speed, affecting tasks ranging from basic operations to complex computations in applications like gaming or video editing.
